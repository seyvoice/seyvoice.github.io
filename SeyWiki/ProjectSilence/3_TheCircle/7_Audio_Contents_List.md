# Unwritten Saga : The Circle 오디오 컨텐츠 목록

게임에 사용되는 배경음악, 효과음, 앰비언스 목록 및 설정.

| Boss_Heroic     | 기사&우정 | 고결한 격돌: 금관, 현, 퍼커션, 남녀합창. 절제된 격렬함.                  | 란슬롯, 아발론기사 전투         |

| Boss_Final      | 모든 테마 | 사가의 종결: 피아노, 첼로, 여성 솔로. 모든 테마가 교차하며 슬픔을 억누르는 공허의 곡. | 모르가나 전투               |
- 기사(메인), 모험, 우정, 마녀 총 4개의 테마를 사용해 전체 배경음악 작곡.
- Stem 연출을 위한 기준, 표현에 대한 연구 필요.

## 3. 앰비언스 목록
| 앰비언스 이름                     | 분류   | 사용 시점                      |
|:----------------------------|:-----|:---------------------------|
| Ambience_Open               | 공간   | 평원, 마을, 도시, 해안. 탁 트인 공간 소리 |
| Ambience_Close              | 공간   | 홀, 방안                      |
| Ambience_Environment_Nature | 환경 | 숲, 늪, 산. 바람과 나무 흔들리는 소리    |
| Ambience_Environment_Fire   | 환경 | 모닥불, 횟불, 불길                |
| Ambience_Environment_Water  | 환경 | 개울, 강, 바다, 호수              |
| Ambience_Environment_Cold   | 환경 | 지하실, 동굴                    |
| Ambience_Weather_Rain       | 날씨   | 비, 폭우                      |
| Ambience_Weather_Storm      | 날씨   | 눈, 눈보라                     |
| Ambience_Threat_Beast       | 위협   | 위험. 주변 짐승 소리               |
| Ambience_Threat_Insect      | 위협   | 공포. 주변 파리와 벌레 소리           |
| Ambience_Crowd_Life         | 군중   | 일상. 주변 생활 말소리              |
| Ambience_Crowd_Chears       | 군중   | 기쁨. 주변 환호                  |
| Ambience_Crowd_Buzz         | 군중   | 혼란. 주변 웅성거림                |
| Ambience_Crowd_Fear         | 군중   | 공포. 주변 공포와 비명              |
- 앰비언스는 공간 + 환경 + 날씨 + 위협/군중 총 4개의 트랙을 교체/믹스하여 장면을 연출한다.
