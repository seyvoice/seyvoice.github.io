# Unwritten Saga 전투 규칙

전투는 D&D 클래식 베이직 룰의 간략 버전을 베이스로 콤보액션을 특징으로 한다.

### ⚔️ 전투 규칙 개요
- **진행**: 턴제. 플레이어가 액션을 선언하면, 다음 차례까지 턴이 진행된다.
- **아군 명령 불가**: 전투는 1:1부터 3:3까지 발생. 아군 행동은 지정 불가능하다.
- **액션**: 플레이어는 턴마다 5가지 액션 중 1가지를 선언한다. 액션마다 턴 수가 다르다.

| 액션  | 기능               | 턴 소모 | 특징                                |
|:----|:-----------------|:-----|:----------------------------------|
| 공격  | 지정한 적에게 피해를 준다   | 1턴   | 차단 가능                             |
| 막기  | 일반 공격을 100% 막는다  | 2턴   | 첫턴에 완벽한방어 가능                      |
| 회피  | 회피확률을 크게 높인다     | 1턴   | 공격, 기술, 아이템에 모두 적용된다              |
| 기술  | 적의 막기를 무시하고 공격한다 | 3턴   | 방어파괴 효과<br/>발동 전후로 모으기, 무방비 턴이 있다 |
| 아이템 | 아이템을 사용한다        | 3턴   | 소모성 아이템의 경우 소모된다                  |

- 공격, 기술, 아이템사용 선언시 대상은 하나만 지정 가능하다.
  - 액션이 남은 상태에서 지정된 대상이 쓰러진 경우, 허공에 수행한다.

### ⚔️ 전투의 핵심은 스턴과 차단
- **완벽한 방어 Perfect Guard**: 막기 첫 턴에 일반공격 명중시 발동. 공격자는 모든 행동이 취소되고 스턴에 빠진다. 공격자은 다음 턴에 새 액션 선언을 가진다.
- **방어 파괴 Guard Break**: 막기 중에 기술 명중시 발동. 방어자는 모든 행동이 취소되고 스턴에 빠진다.
- **차단 Interrupt** :기술 모으기 중에 일반공격 명중시 발동. 방어자는 일정확률로 모든 행동이 취소된다. 추가로 자세회복 턴이 강제된다.
- **스턴** : 일정 턴 동안 행동불능 상태가 된다. 추가로 자세회복 턴이 강제된다.

# 전투 상세 규칙

# 1. 우선권 Initiative
턴마다 우선권 굴림으로 액션순서를 결정한다.
- `우선권 = 2D6 + 액션 우선점수 + 무기 우선점수`
- 우선권이 높은 순서대로 먼저 액션을 수행한다.
  - 2D6이 12일 경우 최우선으로 액션, 2일 경우 최후순

| 액션 우선점수 ||
|:----|:----|
| 막기	 | +3 |
| 기술	 | +2 |
| 아이템	 | -2 |

| 무기 우선점수 ||
|:--------|:---|
| 단검      |+3|
| 검       |0|
| 한손창     |+1|
| 도끼      |-1|
| 둔기      |-2|
| 양손무기    |-4|
| 쌍수무기    |+2|

- 인물에 따라 우선권 보너스가 추가될 수 있다.
  
# 2. 명중 및 회피 Hit
공격/기술 발동시 명중률 굴림으로 명중 여부를 결정한다.
- `명중 조건 : 2D6 ≥ DC:5 + 방어자 회피 보너스`
  - `2D6가 12일 경우 반드시 명중, 2일 경우 반드시 빗나감` 

| 회피 보너스    ||
|:----------|:---|
| 회피중 | +3   |
| 기마중 지상공격에 | 0   |

- 인물에 따라 명중/회피 보너스가 추가될 수 있다.

# 3. 피해량 Damage
명중시 피해 굴림으로 피해량을 결정한다.
- `피해량 = 반올림(2D6 * 공격자 무기 피해) - (방어자 방어력 - 공격자 방어무시)`
  - `(방어자 방어력 - 공격자 방어무시)는 0보다 작을 수 없음`
- 방어자의 HP이 피해량 만큼 감소하며, HP가 0이 되면 대상은 쓰러진다.
  - 사망 여부는 대상과 이벤트에 따라 다르다.

| 무기 피해  ||
|:-------|:---|
| 단검     | 0.5 |
| 검      | 1 |
| 창      | 0.8 |
| 도끼     | 1.2 |
| 둔기     | 0.8 |
| 양손무기   | 1.6 |

| 방어력 예시    || 
|:----------|:----|
| 맨몸       | 0     |
| 사슬갑옷      | 2   |
| 비늘갑옷      | 4   |
| 판금투구&갑옷   | 8   |

| 무기 방어무시   ||
|:----------|:----|
| 둔기        3  |
| 양손무기 | 6   |

- 인물에 따라 피해/방어력/방어무시 보너스가 추가될 수 있다.

# 4. 스턴 Stun
완벽한방어/방어파괴 발생시, 스턴 굴림으로 턴 수를 결정한다.
- `스턴 턴 수 = 반올림(2D6/3)`
- 스턴은 공격자/방어자 양쪽에 발생할 수 있다.
  - 공격자 스턴: 완벽한방어
  - 방어자 스턴: 방어파괴
- 대상의 선언된 모든 액션이 취소되고, 스턴 굴림 만큼의 턴 동안 아무 행동을 할 수 없다.
- 추가로 자세회복 턴이 강제된다.
- 인물에 따라 스턴감소 보너스가 추가될 수 있다.

# 5. 차단 Interrupt
기술 모으기 중 일반공격에 명중시, 차단 굴림으로 차단 여부를 결정한다.
- `차단 조건 : 2D6 >= DC:7`
  - `2D6가 12일 경우 반드시 차단, 2일 경우 반드시 버티기`
- 차단 발생시, 방어자의 모든 예약된 액션이 취소되고, 자세회복 1턴이 강제된다.
- 인물에 따라 차단 버티기 보너스가 추가될 수 있다
