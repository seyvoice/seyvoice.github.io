# Unwritten Saga: The Circle 접근성 옵션

## Base Features:
### Steam Input API:
- 지원: 따라서 엑박/플스 컨트롤러 및 스팀덱 조작 및 사용자 키매핑 지원.
### Adjustable Difficulty:
- 전투 난이도 쉬움/표준/어려움 옵션 제공.
### Save Anytime:
- 프로필 별 자동저장 지원.
- 공용 수동저장 슬롯 지원.
### Custom Volume Controls:
- Master/BGM/SFX/Ambience/Voice 볼륨 각각 옵션 제공.
### Directional Audio:
- stero sound  지원.
## Visual Features:
### Adjustable Text Size:
- 텍스트 폰트 작게/표준/크게 옵션 제공.
- UI 폰트 크기 작게/표준/크게 옵션 제공.
### Subtitle Option, Color Alternatives, Contrast Controls:
- 대사 색상강조/기본표시 옵션 제공.
- 화자표시 색상강조/표시/끄기 옵션 제공.
- 텍스트 색상 검정/하양 옵션 제공.
- 텍스트 바탕색 어둡게/표준/밝게 옵션 제공.
- UI 바탕색 어둡게/표준/밝게 옵션 제공.
- 아이콘 중심 UX로 색 없어도 플레이 가능.
### Camera Comfort:
- 적 흔들림 강/약/끄기 옵션 제공.
- 내 캐릭터시트 흔들림 강/약/끄기 옵션 제공.
## Input Features:
- 마우스만으로 조작 가능.
- 키보드만으로 조작 가능.
- 터치만으로 조작 가능.
- 컨트롤러만으로 조작 가능.
## Time Input:
- 입력에 시간을 요구하지 않음.
